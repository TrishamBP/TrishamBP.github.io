3D Convection in an Electronic Box.

Problem Statement: Currently, a design for an electronics box calls for 11 electronic boards and a battery to be enclosed. Each electronic component will generate 2.1 watts of heat. The current design uses two fans to blow air through the box to cool the electronics. The fans blow air at a rate of 35 cubic feet per minute. The box is expected to operate in an environment where the temperature reaches a maximum of 114 degrees Fahrenheit. The main constraint on the design is that the electronic components need to be maintained below 200 degrees Fahrenheit, otherwise the components could be damaged. The box itself is ABS plastic.

Properties of materials.

Methodology/Steps:

A.Pre-Analysis and Setup:Open ANSYS Workbench and load Fluid Flow or ANSYS-Fluent.

B.Geometry:The geometry for this box was designed in Solidworks and consists of 11 electronic boards, 2 inlet and outlet holes, 16 connection holes and 6 side wall panels. Import the geometry in workbench.The next step is to create a flow geometry. The air is going to flow through the inside of the box; however, we need a geometry for this empty space for the software to be able to calculate the solution. We accomplish this by creating Caps, then using those caps to create Fill. Next, we need to create fill geometry. Go to Tools > Fill. Name the fill, Flow Geometry. Also, change the Extraction Type. Select Extraction Type > By Caps. Select all the bodies and form a new part, then connect the geometry to the mesh.

C.Mesh: Created named selection by selecting the outlet and inlet holes and name them outlet and inlet. Selected the surface behind the front surface, electronic boards and the battery, then named them as Heat Generation. (As they are responsible for heat generation). Generated the mesh using default element size.

D.Physics: Double click on setup to launch ANSYS-Fluent. The first thing that needs to be done is to tell Fluent to solve for temperature. To do this, we need to turn on the Energy Equation. Create materials air copper and ABS with the values given in the table. In the Boundary Conditions window, select Inlet. Check to make sure that the Type has correctly defaulted to Velocity-Inlet and press Edit. In the Momentum Tab, we will specify the velocity of the incoming air. Knowing that the inlet feed rate is 35 ft3/min and that the diameter of the holes is 4.72 inches, we can solve for the inlet speed. The ambient air is 114 degrees Fahrenheit. Fluent only accepts temperatures in Kelvins. 114 degrees Fahrenheit = 318.706 Kelvins. The Backflow Total Temperature to 318.76 Kelvin. In the Boundary Conditions menu, select wall-solid and press Edit. Select the Thermal tab, and change the Material to ABS. Under Zones, select Heat Generation and press Edit. Change the material name to copper. Enter the heat generation as 40682 w/m3. In the Outline, select Reference Values. Underneath Compute From, select Inlet.

Conclusion: From the temperature profile of the electronic board from the temperature of the electronic board, it can be concluded that the maximum temperature of the electronic components is 336K, which is lower than the maximum limit of 200F (366.483K), so it can be concluded that the electronic components will not be damaged and that the air circulating fan system works properly for the electronic board box.
